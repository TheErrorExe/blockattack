Wie installiere ich Java?

Um den Fabric-Installer auszuführen, benötigst du Java. Hier erfährst du, wie du es installierst:

  1. Überprüfen, ob Java bereits installiert ist:

    1. Drücke Windows + R, um das „Ausführen“-Fenster zu öffnen.

    2. Gib cmd ein und klicke auf „OK“.

    3. Gib den Befehl java -version ein und drücke „Enter“.

    4. Wenn eine Versionsnummer erscheint (z.B. „Java 17.0.x“), ist Java bereits installiert.

  2. Java installieren (falls nicht vorhanden):

    1. Gehe auf die Java-Download-Seite: Java SE Downloads.

    2. Wähle die neueste Version und lade das Installationspaket für Windows herunter.

    3. Führe das Installationsprogramm aus und folge den Anweisungen, um Java zu installieren.

Wie installiere ich Fabric?

Nun kannst du den Fabric-Installer verwenden, um Fabric für Minecraft zu installieren:

  1. Fabric Installer herunterladen: Besuche Fabric und lade den Installer herunter.
  2. Fabric installieren: Öffne den Installer, wähle die Minecraft-Version und klicke auf „Installieren“.
  3. Minecraft starten: Öffne den Minecraft Launcher, wähle das „Fabric“-Profil und klicke auf „Starten“.

Wie installiere ich die Fabric API?

Viele Mods, darunter auch Simple Voice Chat, erfordern die Fabric API. Folge diesen Schritten:

  1. Fabric API herunterladen: Gehe zu Fabric API auf CurseForge und lade die neueste Version herunter.
  2. Fabric API installieren: Verschiebe die heruntergeladene fabric-api.jar-Datei (nur wenn da api drinne ist im namen) in den „mods“-Ordner von Minecraft.
  3. Minecraft Ordner finden: Drücke Windows + R, gib %appdata% ein und klicke auf „OK“. Öffne den Ordner „.minecraft“ und dann den Ordner „mods“.

Wie installiere ich Simple Voice Chat?

Jetzt kannst du den Simple Voice Chat Mod installieren:

  1. Simple Voice Chat Mod herunterladen: Gehe zu Simple Voice Chat auf CurseForge und lade die neueste Version herunter.
  2. Mod installieren: Drücke Windows + R, gib %appdata% ein und klicke auf „OK“. Öffne den Ordner „.minecraft“ und dann den Ordner „mods“. Ziehe die heruntergeladene .jar-Datei (für simple voice chat, kein fabric) in den „mods“-Ordner von Minecraft.
  3. Minecraft starten: Starte Minecraft mit dem Fabric-Profil. Der Voice Chat sollte nun verfügbar sein!

Hinweis: Bedrock-Spieler können den Voice Chat nicht nutzen.
